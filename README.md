# KazMentor Unity Game

Interactive educational game with ChatGPT NPCs and interactive quests.

## Highlights
- AI‑driven dialogues (ChatGPTManager.cs)  
- Drag‑and‑drop circuit puzzles  
- Multiple scenes (Lab, School, Museum)

## Requirements
- Unity 2021.3 LTS+  
- Input System, UI Toolkit, TextMesh Pro

## Run
1. `git clone https://github.com/adilkhan-tubulbayev/KazMentor.git`
2. Open `KAZMENTOR` folder in Unity  
3. Add your OpenAI API key in ChatGPTManager  
4. Play **MainMenu** scene
